# py_daemon.py

## Usage

Run py_daemon.py.

    $ ./py_daemon.py

Two files will appear in your home dir.

* ~/py_daemon.pid

    Pid of daemon process.

* ~/py_daemon.txt

    Output of daemon process. You will see the date is updated every 10 seconds.
